## 项目简介

由于本人是个独立开发者，经常会从事一些计算机专着的写作与翻译工作（主要作品可参考我的[豆瓣主页](https://site.douban.com/217670/)），业余偶尔也会有一些机会定期或不定期地参与国内外大学、开源社区中的一些个人研究项目，也帮忙指导过一些硕士论文，所以在编程语言研究、Web 应用程序开发等领域累积了一定的专业知识和实践经验。本项目是我基于[卡片笔记写作法]()建立的个人知识库项目，主要作用如下：

- **知识管理**：通过双链笔记的方式，将我在学习、研究、写作、翻译、开发等过程中产生的各种零散知识进行整理，以便更好地管理和使用这些知识，提升自己之后的写作和开发效率。
- **知识分享**：将整理好的笔记内容发布到 GitHub 上，以便与更多的读者分享我的学习经验和心得。

## 阅读建议

由于这是一个基于双链功能建立的笔记库项目，建议读者使用 [Obsidian 笔记软件](https://obsidian.md/)或者安装了 [Foam 插件](https://foambubble.github.io/foam/)的 VS Code 编辑器来阅读该笔记库中的内容，以便更好地体验双链笔记的优势。
