            \noindent
            勾股定理，是一个基本的几何定理，指直角三角形的两条直角边的平方和等于斜边的平方。中国古代称直角三角形为勾股形，并且直角边中较小者为勾，另一长直角边为股，斜边为弦，所以称这个定理为勾股定理，也有人称商高定理，其几何关系如图\ref{fig:勾股定理示意图}所示。

            在这里，我们使用了 wrapfig 宏包提供的 wrapfigure 环境，该环境会自动将图片元素放置在右侧，并让文字环绕在图片的左侧。
