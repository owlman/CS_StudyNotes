## 二、前 AGI 时代的 AI 研究（1950s–2010s 中期）

### 2.1 符号主义与专家系统（1950s–1980s）

- 逻辑推理与知识表示（谓词逻辑、产生式规则）
- 专家系统（DENDRAL、MYCIN、XCON）
- 局限性：知识获取瓶颈、脆性、组合爆炸

### 2.2 统计方法与贝叶斯复兴（1980s–2000s）

- 概率图模型（贝叶斯网络、HMM、CRF）
- 从手工知识到数据驱动的范式转变
- 典型应用：语音识别、机器翻译（统计 MT）、信息检索

### 2.3 机器学习的崛起（1990s–2010s）

- 监督学习 / 无监督学习 / 强化学习三大范式确立
- 核方法（SVM）与集成方法（随机森林、GBDT）
- "特征工程" 作为核心竞争力

### 2.4 深度学习的突破（2012–2017）

- AlexNet（2012）与 ImageNet 时刻
- 表征学习：不再手工设计特征
- 关键架构：CNN（视觉）、RNN/LSTM（序列）、GAN（生成）
- 核心驱动力：大数据 + GPU + 更好的优化算法

### 2.5 强化学习的独立发展线

- 从 TD-Gammon 到 DQN（2013/2015）
- AlphaGo（2016）的里程碑意义
- 模型无关的通用 RL 方法：策略梯度、Actor-Critic

## 三、AGI 时代的 AI 研究方法（2017 至今）

### 3.1 Transformer 架构的范式革命（2017）

- 自注意力机制与并行化优势
- 从 "Attention Is All You Need" 到统治性架构

### 3.2 预训练范式的确立（2018–2022）

- BERT：掩码语言模型与双向上下文
- GPT 系列：自回归语言模型与涌现能力
- 规模化定律（Scaling Laws）：参数、数据、算力的可预测关系
- 预训练 → 微调（Fine-tuning）的默认工作流

### 3.3 对齐与可控性（2022 至今）

- 指令微调（Instruction Tuning）：让模型"听懂人话"
- RLHF（人类反馈强化学习）：InstructGPT / ChatGPT
- 直接偏好优化（DPO）等对平替方法
- 红队测试与安全对齐

### 3.4 工具使用与 Agent 化

- Function Calling 与工具增强
- 检索增强生成（RAG）及其变体
- 多智能体协作（AutoGen、CrewAI 等框架）
- 规划与推理：Chain-of-Thought、Tree-of-Thoughts

### 3.5 多模态与统一架构

- 视觉-语言模型（CLIP → GPT-4V → 统一多模态）
- 文生图/视频（Stable Diffusion、Sora）
- 迈向通用感知与控制

## 四、理解 AGI 的能力边界

### 4.1 当前的确定边界

- 幻觉（Hallucination）：根源于统计生成，非事实检索
- 因果推理的缺失：关联 ≠ 因果
- 长程一致性与规划能力的局限
- 样本效率极低：人学一次，模型需万亿 token

### 4.2 理论层面的约束

- 没有免费午餐定理（No Free Lunch）
- 计算复杂度与可学习性（PAC 学习框架）
- 哥德尔不完备与形式系统内的自我指涉

### 4.3 工程与社会的边界

- 能源与算力可持续性
- 数据版权与隐私
- 价值对齐的根本困难（价值观多元性）
- 评估危机：基准饱和与 Goodhart 定律

### 4.4 什么可能仍是禁区

- 真正的世界模型与物理常识
- 主观体验（意识问题）与 Qualia
- 创造性理解 vs 统计重组

## 五、研究方法论的演变规律与启示

### 5.1 三大转向

- 从知识驱动 → 数据驱动
- 从专用系统 → 通用基础模型
- 从独立组件 → 端到端学习

### 5.2 不变的核心问题

- 泛化（Generalization）始终是第一性原理
- 表示（Representation）决定了能力的上限
- 数据质量 >> 模型大小（在同等算力约束下）

### 5.3 对学习者的建议

- 追溯思想源头，而非追逐 SOTA
- 动手验证关键论文的核心假设
- 在学习路线图各阶段中交叉印证
