# Flowchart Precision

## Design Philosophy

**Aesthetic**: Technical diagrammatic precision meets minimalist Swiss design.

**Form & Space**: Clean geometric shapes with precise spacing. Every node occupies its space with intention. Flowcharts breathe through calculated negative space—never cramped, never wasteful. Each box, arrow, and label exists in deliberate spatial relationship to its neighbors. The composition follows grid logic: 8px or 16px baseline grid, elements snap to intersections, whitespace ratios create visual rhythm.

**Color & Material**: A restrained palette where color serves as information layer. Primary workflow nodes use cool technical blues (#3B82F6, #60A5FA). Decision points and human roles use warm accents (#F59E0B, #FBBF24). Feedback loops employ complementary purples (#8B5CF6). Background remains neutral off-white (#F8FAFC). Edges and connectors in muted slate (#64748B). The palette suggests a professional documentation system—clinical but not cold.

**Scale & Rhythm**: Nodes scale proportionally to information density. Wide nodes for multi-step processes, tall nodes for single concepts. Consistent corner radius (8px for small, 12px for large) creates visual family. Typography remains whisper-quiet: 11px labels, 13px phase headers, 14px main title. Text never competes with geometry.

**Composition & Balance**: Top-to-bottom primary flow, left-to-right secondary connections. Main trunk flows vertically with generous spacing. Branch nodes offset horizontally. Decision diamonds create visual punctuation. The eye naturally follows the main artery while peripheral branches remain accessible but subordinate.

**Visual Hierarchy**: Three tiers—Phase headers (largest, bold), Process nodes (medium, regular), Labels (smallest, muted). Arrows carry directional meaning: solid for primary flow, dashed for feedback loops, dotted for optional paths. The hierarchy makes complex workflows scannable in seconds.

**Information Architecture**: This philosophy treats the flowchart as a precision instrument. Like a circuit diagram or architectural blueprint, the visual language communicates function through form. Color encodes meaning rather than decoration. The result feels like technical documentation from a design-obsessed company—every pixel purposeful, every spacing decision reasoned.
